## 1️⃣ What is the difference between var, let, and const?

I'll keep it simple. All three are used to declare variables, but var's scope is at the function level, while let and const's scope is at the block level. Const means that once the variable is set, its value cannot be changed, whereas let's allow it to be changed, but that also works at the block level.

## 2️⃣ What is the spread operator (...)?

The Spread Operator is a JavaScript feature that makes it easy to separate or expand the elements of an array or object, such as adding new values, copying them, or using them in different ways.

---

## 3️⃣ What is the difference between map(), filter(), and forEach()?

These are the three array methods in JavaScript. The main differences are:

map(): It creates a new array by applying a function to each element.

filter(): It uses a function to return only those elements that meet a condition, i.e. it filters the array.

forEach(): It runs a function on each element, but it does not return a new array; it only runs the function on each element.

---

## 4️⃣ What is an arrow function?

Arrow function is a short and simple syntax in JavaScript that allows you to create an anonymous function with less code than defining a regular function.

---

## 5️⃣ What are template literals?

Template literals are a feature of JavaScript that allows you to write strings using backticks (`), and you can insert variables or expressions there using ${}, so strings can be more flexible and even multi-line.
